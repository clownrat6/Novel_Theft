# 轻文解析器

爬取轻小说，并打包成 EUPB，插图只嵌入开头。

## 用法

python main.py --task [main/picture]

>--task main 为将 txt 以及图片 url 解析并存放至 txt 中。  
>--task picture 为下载图片 url 并存放到对应文件夹。
